High-level Architecture

DISCLAIMER

This architecture and the demonstration is intended to showcase the "art-of-the-possible" as it relates to Genesys Cloud. In this case, leveraging the Genesys Cloud Open Messaging API in combination with serverless resources in AWS, presents one of many potential approaches to implementing messaging capability between users and agents without passing or sending the users' IP Address information to Genesys Cloud services. This architecture and technical demonstration is not intended for production purposes but can serve as an example of how to implement this same concept or functionality for your use case(s).
